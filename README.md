Um sistema web institucional desenvolvido para o **Dona Marlene Hotel**, localizado no coração da Amazônia, na cidade de Codajás (AM). O projeto tem como objetivo principal apresentar a infraestrutura do hotel, detalhar as opções de acomodação e facilitar o contato direto dos hóspedes para reservas.

🌐 **Acesso ao projeto em produção:** [Dona Marlene Hotel](https://donamarlenehotel.azurewebsites.net/)

## 🚀 O Projeto

A aplicação foi construída com o objetivo de fornecer uma navegação rápida, intuitiva e totalmente responsiva. Explorando o ecossistema da Microsoft para desenvolvimento web, o sistema utiliza o padrão arquitetural **ASP.NET Core MVC**, garantindo um backend sólido, escalável e de fácil manutenção para futuras atualizações estruturais. O deploy contínuo e a hospedagem foram realizados nos serviços de nuvem do **Microsoft Azure**, assegurando a estabilidade da aplicação.

### ✨ Funcionalidades Principais

* **Página Inicial (Landing Page):** Apresentação dos diferenciais do hotel, como o sossego, conforto e a localização privilegiada.
* **Catálogo de Acomodações:** Exibição das opções de quartos (Solteiro, Casal+ e Família) e suas respectivas comodidades (Ar-condicionado, TV a cabo, Wi-Fi, Frigobar).
* **Integração Direta com WhatsApp:** Botões de "Call-to-Action" (CTA) parametrizados que redirecionam o usuário diretamente para o WhatsApp do hotel, já preenchendo a mensagem com o quarto de interesse.
* **Galeria de Imagens:** Seção visual dedicada a apresentar a infraestrutura física (fachada, garagens, corredores).
* **Design Responsivo:** Interface otimizada para funcionar perfeitamente em dispositivos móveis (mobile-first) e desktops.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** C#
* **Framework:** .NET / ASP.NET Core MVC
* **Frontend:** HTML, CSS
* **Cloud / Hospedagem:** Microsoft Azure (Azure App Service)
